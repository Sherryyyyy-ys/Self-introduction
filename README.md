# Sherry Ding Premium Portfolio

A premium personal portfolio website built with Next.js, React, Tailwind CSS, and Framer Motion.

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm run start
```
